Abstract

Diffusion Large Language Models (dLLMs) represent a promising paradigm beyond autoregressive modeling, enabling flexible and parallel token generation. However, their uniform computation across all transformer blocks introduces unnecessary overhead during inference. In this work, we present dMoE, a framework that introduces Learnable Block Experts into dLLMs. dMoE adds block-level Mixture-of-Experts routing on top of LLaDA-2.0-mini, enabling the model to adaptively activate a subset of transformer blocks for each input during the diffusion denoising process. This plug-and-play design requires no architectural changes to the base model and achieves a favorable efficiency-accuracy trade-off. Extensive evaluations demonstrate that dMoE achieves competitive performance on reasoning and general benchmarks including GSM8K, MATH500, MMLU, and ARC-C, while reducing unnecessary computation through adaptive expert activation.
